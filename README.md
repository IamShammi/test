# DevOps Interview Case Study: Expense Accounting System

## Overview
This is a technical interview case study for a DevOps Engineer position. The project contains three implementations of a simple expenses accounting calculator API.

## Structure
```
├── docker-compose.yml          # PostgreSQL and Redis containers
├── schema.sql                  # Database schema
├── java-expenses/              # Java Spring Boot implementation
├── dotnet-expenses/            # .NET Core implementation
├── python-expenses/            # Python Flask implementation
└── terraform/                  # AWS EC2 deployment configuration
```

## Quick Start
1. Choose your preferred language implementation (Java, .NET, or Python)
2. Start PostgreSQL and Redis container
3. Navigate to the chosen implementation directory
4. Follow the README in that directory to run the application

## Interview Context
- Tech stack: Postgres, Redis, Terraform, EC2
